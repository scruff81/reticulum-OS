# reticulum-OS
a reticulum network OS based on Debian for raspberrypi pi
Based on debian 13 (trixie). included is rnsd, lxmf, and a WiFi hotspot. Rnsd starts on boot along with the WiFi hotspot. For propogation nodes lmxd-p starts on boot as well. config files for rnsd and lxmf are in the standard directories. This OS only includes RF interfaces, rnode and KISS for digirig mobile are configured. interfaces will have to be enabled through the config file and the pi will need to be rebooted for first use. Refer to reticulum network website for additional interfaces and modifications to config files
