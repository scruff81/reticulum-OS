# reticulum-OS
a reticulum network OS based on Debian for raspberry pi
Based on debian 13 (trixie). Included is rnsd, lxmf, nomadnet, direwolf, and a WiFi hotspot. lmxd-p starts on boot after the wifi hotspot starts. config files for rnsd and lxmf are in the standard directories. This OS only includes RF interfaces, rnode and KISS for digirig mobile are configured. Interfaces will have to be enabled through the config file and the pi will need to be rebooted for first use. Refer to reticulum network website for additional interfaces and modifications to config files.
